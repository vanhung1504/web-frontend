Bài tập 1

Chuyển đổi tiền tệ

Viết hàm USD2VND(n), hàm nhận một tham số (input) n là số tiền USD, tính và in ra số tiền tương ứng theo mệnh giá Việt Nam theo công thức:

VND = USD * 23.500

My code

let USD2VND = function (usd) {
  console.log(`Quy đổi: $${usd} = ${usd * 23500} VNĐ`);
};
USD2VND(1);
USD2VND(5);
USD2VND(10);

Mời bạn mở cửa sổ console trên trình duyệt để xem kết quả!

Bài tập 2

Tính lãi suất kép ngân hàng (kỳ quan thứ 8)

Viết hàm compoundInterest(P, r, n, t), hàm nhận 4 tham số (input):

  • P: số tiền gốc ban đầu
  • r: lãi suất hàng năm
  • n: số lần ghép lãi 1 năm (*)
  • t: số năm gửi ngân hàng

(*) Ví dụ gửi theo tháng thì 1 năm ghép lãi 12 lần => n = 12, gửi theo quý (3 tháng) thì 1 năm ghép lãi 4 lần => n = 4, gửi theo năm thì 1 năm ghép lãi 1 lần => n = 1, …

Công thức:

A = P * (1 + r / n)**(n * t)

My code

let compoundInterest = function (P, r, n, t) {
  console.log(
    `Với số tiền gốc ban đầu là ${P} VNĐ, lãi suất ngân hàng ${
      r * 100
    }% một năm, ghép lãi ${n} lần/năm thì tổng số tiền bạn nhận được sau 
    ${t} năm gửi ngân hàng là: ${
      P * (1 + r / n) ** (n * t)
    }VNĐ`
  );
};
compoundInterest(2000000000, 0.09, 12, 3);
compoundInterest(100000000, 0.065, 1, 1);
compoundInterest(50000000, 0.05, 2, 2);

Mời bạn mở cửa sổ console trên trình duyệt để xem kết quả!
